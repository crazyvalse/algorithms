### promise

Promise 是一个对象，它表示一个异步操作的最终完成或者失败。。由于它的 then 方法和 catch/finally 方法会返回一个新的 promise 所以可以允许我们链式调用，解决了传统的回调地狱问题。

- Promise 的状态一经改变就不能再改变；
- .then 和 .catch 都会返回一个新的 Promise；
- catch 不管被连接到那里，都能捕获上层未捕获过的错误；
